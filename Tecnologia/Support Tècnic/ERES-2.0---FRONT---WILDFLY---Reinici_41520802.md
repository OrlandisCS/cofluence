**Aturar ERES 2.0**

/etc/init.d/wildfly\_aoc stop

**Iniciar ERES 2.0**

/etc/init.d/wildfly\_aoc start

**Veure procés**

ps aux | grep -i wild

**Matar procés**

kill \[Num pid\]

#En cas que amb el kill no mori el procés fer un:

kill -9 \[Num pid\]
